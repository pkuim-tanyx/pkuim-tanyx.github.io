Convex Set#

Def. A set $S$ is convex iff $\forall x,\ y\in S$, $\lambda x+(1-\lambda)y\in S$. (Imagine a line that connects point x and y)

Prop. If $A,\ B$ is convex, then $A\cap B$ and $A+B:=\lbrace x+y\ |\ x\in A,\ y\in B\rbrace$ are convex. Note that $A\cup B$ is not necessarily convex.

Def. A set $S$ is called subspace iff $\forall x,\ y\in S$, $a,\ b\in\mathbb{R}$, $ax+by\in S$.

e.g. The null space and range (column) space of a matrix $A\in\mathbb{R}^{m\times n}$:

  • $\mathcal{N}(A)=\lbrace x\ |\ Ax=0\rbrace$,

  • $\mathcal{R}(A^\top)=\lbrace A^\top z\ |\ z\in\mathbb{R}^m\rbrace$.

Note that $\text{dim}(\mathcal{N}(A))=n-\text{rank}(A)$, $\text{dim}(\mathcal{R}(A^\top))=\text{rank}(A)$, and $\mathcal{N}(A)\perp\mathcal{R}(A^\top)$, hence $\mathcal{R}(A^\top)$ is the orthogonal complement of $\mathcal{N}(A)$.

e.g.2 In multivariable OLS, we know the best estimator of coefficient is

$$ \hat{\beta}=(X^\top X)^{-1}X^\top y \implies X\hat{\beta}=\underbrace{X(X^\top X)^{-1}X^\top}_{\text{projection matrix}} y:=\Pi y $$

(Recall the definition of a projection matrix: $\Pi^\top =\Pi$ and $\Pi^2=\Pi$)

Therefore, the residual can be calculated as

$$ \hat{\epsilon}=y-X\hat{\beta}=(I-\Pi)y \implies \Pi\hat{\epsilon}=0 $$

, namely residual belongs to the null space of projection matrix, and it is orthogonal to the linear space generated by feature matrix.

Def. An affine space $S_0$ is generated by translating a subspace $S$ according to vector $x_0$, namely $S_0=\lbrace x+x_0:x\in S\rbrace$. (Note that 0 must be concluded into any subspace according to its definition)

Obviously, a solution set of linear equations $Ax=b$ with $b\neq 0$ is an affine space.


Def. A set is called hyperplane iff $S=\lbrace x\ |\ a^\top x=b\rbrace$, and A set is called halfspace iff $S=\lbrace x\ |\ a^\top x\geq b\rbrace$. ($a\neq 0$)

Corollary: The intersection of several halfspaces is called a polyhedron, namely $P=\lbrace x\in\mathbb{R}^n\ |\ Ax\geq b,\ A\in\mathbb{R}^{m\times n}\rbrace$.

Def. The convex hull of a set of points $\lbrace x_i\rbrace_{i=1}^n$ is all the convex combination of point set, namely

$$ \text{conv}\lbrace x_i\rbrace_{i=1}^n=\left\lbrace \sum_{i=1}^n\lambda_ix_i\ \Big|\ \sum_{i=1}^n\lambda_i=1,\ \lambda_i\geq 0\right\rbrace $$

Def. A set $C$ is called a cone iff $x\in C\implies \lambda x\in C,\ \forall\lambda\geq0$. Furthermore, the conic hull of a set of points $\lbrace x_i\rbrace_{i=1}^n$ is all the conic combination of point set, namely

$$ \text{cone}\lbrace x_i\rbrace_{i=1}^n=\left\lbrace \sum_{i=1}^n\lambda_ix_i\ \Big|\ \lambda_i\geq 0\right\rbrace $$

Prop. The convex/conic hull of set $\lbrace x_i\rbrace_{i=1}^n$ is the smallest convex set/cone that contains $\lbrace x_i\rbrace_{i=1}^n$.

※Thm. (Minkowski) Any polyhedron can be represented as the convex combination of its all vertices and the conic combination of its all extreme directions if it is unbounded.


Some examples of different cones:

  • Orthant cone $C=\lbrace (x_1,\cdots,x_n)\ |\ x_i\geq 0\rbrace$;

  • Second order cone $C=\lbrace (x_1,\cdots,x_n)\ |\ ||(x_1,\cdots,x_{n-1})||_2\leq x_n\rbrace$;

  • Semidefinite cone $C=\lbrace A\ |\ A^\top =A,\ x^\top Ax\geq 0\rbrace$;

  • Polyhedral cone $C=\lbrace x\ |\ Ax\geq0\rbrace$.

For a bounded polyhedron $P=\lbrace x\ |\ Ax\geq b\rbrace$, its recession cone is defined as the set of all recession directions, namely $C=\lbrace d\ |\ A(x+\lambda d)\geq b\rbrace=\lbrace d\ |\ Ad\geq 0\rbrace$.


Thm. (Seperate hyperplane theorem) Let $S\subseteq\mathbb{R}^n$ be a nonempty convex set, $x^\in\mathbb{R}^n$ and $x^\notin S$. Then there must exists some vector $d\in\mathbb{R}^n$ and $\delta\in\mathbb{R}$ such that

$$ d^\top x\geq \delta,\ \forall x\in S;\quad \text{meanwhile }d^\top x^*<\delta $$

（i.e. The hyperplane denoted by normal vector $d$ can strictly seperate $x^*$ and $S$）

Thm. (Farkas’ Lemma) For $A\in\mathbb{R}^{m\times n}$, $b\in\mathbb{R}^m$, then exactly one of the two following statements will hold:

  • $\exists x\geq 0$ such that $Ax=b$;

  • $\exists p$ such that $p^\top A\geq 0^\top$ and $p^\top b<0$.

(Geometric explain: consider the conic hull generated by $A$’s column vectors. Statement 1 implies that $b$ is in the conic hull, otherwise there must exist a hyperplane to seperate $b$ and the conic hull)

Thm. (Supporting hyperplane theorem) For any nonempty convex set $S$ and $x_0\in\partial S$, there always exists a supporting hyperplane $H=\lbrace x\ |\ a^\top (x-x_0)=0\rbrace$ satisfying $a^\top(x-x_0)\leq0,$ $\forall x\in S$. (similar to the tangent of convex function)

Polyhedron and Linear Programming#

general form of LP (the constraint can be described as a polyhedron):

$$ \begin{gather*} \min\ c^\top x\ \text{s.t. }Ax\geq b\ \end{gather*} $$

standard form of LP (can be transformed by introducing slack variables):

$$ \begin{gather*} \min\ c^\top x\ \text{s.t. } Ax=b\ x\geq 0 \end{gather*} $$

To dig into the properties of polyhedron, we introduce some concepts:

  • a point $x\in P$ is called vertex if $\exists c\in\mathbb{R}^n$ such that $c^\top x<c^\top y$ for all $y\in P$ and $y\neq x$. (algebraic property)

  • a point $x\in P$ is called extreme point if we cannot find $y,z\in P$ (both diffrent from $x$) and $\lambda\in[0,1]$ such that $x=\lambda y+(1-\lambda)z$. (geometric property)

  • a ray $d$ is called extreme ray if it cannot be expressed as the combination of two directions, namely it is the “boundary of recession cone” .

※Thm. Suppose that the polyhedron $P=\lbrace x\ |\ a_i^\top x\geq b_i\rbrace,\ i=1,2,\cdots,m\rbrace$ is nonempty. Then, the following statements are equivalent:

  • The polyhedron $P$ has at least one extreme point;

  • The polyhedron $P$ does not contain a line;

  • $\text{rank}(\lbrace a_i\rbrace_{i=1}^m)=n$.

Corollary: (Caratheodory) Let $P$ be the convex hull of matrix $A$’s columns, then all elements of $P$ can be expressed as the convex combination of $A$’s columns with at most $m+1$ nonzero coefficiences. (hint: construct a new polyhedron and use the theorem above)

※Thm. For a general LP, the optimal value is either $-\infty$ or obtained at an extreme point. If the optimal value is $-\infty$, then there must exist an extreme ray such that $c^\top d\leq 0$ (so that you can continuously decrease the objective function without violating constraints).


Def. A constraint is called active/binding on $x^*$ if all the equality holds.

Def. A point $x^*\in\mathbb{R}^n$ is called basic solution (BS) if

  • all equality constraints are active;

  • out of the active constraints, there are $n$ of them that are linearly independent.

Especially, if a BS satisfies all constraints, then it is called basic feasible solution (BFS).

We propose BFS to seek for the optimality because it is relevently easy to obtain.

Def. A BS is called degenerated on $x^\in\mathbb{R}^n$ if there are more than $n$ active constraints on $x^$.

Thm. $x\text{ is a vertex}\iff x\text{ is an extreme point}\iff x\text{ is a BFS}$

  • If $\bar{c}\geq 0$, then $x^*$ is optimal;
