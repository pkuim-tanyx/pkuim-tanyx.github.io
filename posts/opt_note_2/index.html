Convex Functions#

Def. A function $f:S\subseteq\mathbb{R}^n\rightarrow\mathbb{R}^n$ is called convex iff we have $\lambda\in[0,1]$ and different $x_1,\ x_2\in S$ such that

$$ f(\lambda x_1+(1-\lambda)x_2)\leq \lambda f(x_1)+(1-\lambda)f(x_2) $$

otherwise, $f(x)$ is called concave. $f(x)$ is convex $\iff -f(x)$ is concave.

e.g. all $\ell_p$ norms ($\ell_p(x):=\sqrt[p]{\sum_{i=1}^n x_i^p}$) are convex, and $||x||_1\geq||x|_2\geq||x||_{\infty}$.

matrix norms:

  • 3 properties: nonnegativity, homogeneity, triangle inequality

  • Frobenius norm: $||A||_F=\sqrt{\sum_i\sum_j A_{ij}^2}=\sqrt{\text{tr}(A^\top A)}$

  • induced norm: $||A||_p=\max\limits_{||x||_p=1}||Ax||_p$ ($p\in\mathbb{N}^*$)

    • $p=1$: $||A||_1=\max_j\lbrace\sum_i|A_{ij}|\rbrace$

    • $p=2$：The maximal singular value of $A$ (namely $\sqrt{\lambda_{\max}(A^\top A)}$)

    • $p=\infty$: $||A||_\infty=\max_i\lbrace\sum_j|A_{ij}|\rbrace$

Def. For a norm $||\cdot||_p$, its dual norm is defined as

$$ ||y||_p^{*}=\sup_{||x||_p\leq 1}x^\top y \textcolor{blue}{=||y||_q\quad \left(\dfrac{1}{p}+\dfrac{1}{q}=1\right)} $$

(This property can be proved by Hölder inequality: $x^\top y\leq ||x||_p||y||_q$ where $pq=p+q$)

Prop. (Jensen’s Inequality) If $f$ is convex and $\sum_{k=1}^n \theta_k=1$, then

$$ f\left(\sum_{k=1}^n \theta_kx_k\right)\leq \sum_{k=1}^n \theta_kf(x_k) $$

Def. For a real-valued function $f:\mathbb{R}^n\rightarrow \mathbb{R}$, its conjugate function is defined as $f^{*}(z)=\sup\limits_{x\in\mathbb{R}^n}z^\top x-f(x)$. Note that $f^{*}$ is always convex, lower semicontinuous, and $f^{**}=f$.


Def. $\nabla f(x)=\left(\dfrac{\partial f(x)}{\partial x_1},\cdots,\dfrac{\partial f(x)}{\partial x_n}\right)$, $H_f(x)[i;j]=\dfrac{\partial^2 f(x)}{\partial x_i\partial x_j}$

Thm. (first order and second order condition)

  • If $f(x)\in\mathcal{C}^1$, then $f(x)$ is convex $\iff f(y)\geq f(x)+\nabla f(x)^\top(y-x)$, $\forall x,y\in\text{dom}f$.

  • If $f(x)\in\mathcal{C}^2$, then $f(x)$ is convex $\iff H_f(x)\succeq 0$ for all $x\in\text{dom}f$.

e.g. variance function $f(x)=\text{Var}(\xi^\top x)$ where $\xi^\top\mathbf{1}=1$ is convex by second order condition.

Def. Given $f(x):S\subseteq \mathbb{R}^n\rightarrow \mathbb{R}^n$ is a real-valued function, then the epigraph of $f(x)$ is defined as $\text{epi }f:=\lbrace (x,t)\ |\ f(x)\leq t\rbrace$.

※Prop. $f(x)$ is convex $\iff \text{epi }f$ is convex

e.g. prove that $f(x)=\ln\left(\sum\limits_{i=1}^n p_i\exp(a_ix_i+b_i)\right)$ is convex.

Some transportation that preserving convexity:

  • nonnegative linear combination: $f_1,\ f_2$ convex $\implies af_1+bf_2$ convex

  • pointwise maximum: $f_1,\ f_2$ convex $\implies \max\lbrace f_1, f_2\rbrace$ convex

  • affine map: $f$ convex $\implies g=f(Ax+b)$ convex

  • $f(x,y)$ convex, $C$ is a convex set $\implies g(x)=\inf\limits_{y\in C}f(x,y)$ convex

  • perspective function: $f$ convex $\implies g(x,t)=tf\left(\dfrac{x}{t}\right)$ convex

e.g. fractional linear function is convexity-preserving, namely $f$ convex $\implies f\left(\dfrac{a^\top x+b}{c^\top x+d}\right)$ convex.


Def. For $x\in\text{dom}f$, if $f(x_0)\leq f(x),\ \forall x\in N_\varepsilon(x_0)$, then $x_0$ is a local minimum.

Thm. If $f$ is a convex function, then any local minimum equals to global minimum.

Def. For a real-valued function $f:\mathbb{R}^n\rightarrow \mathbb{R}$,

  • if $\forall x,y\in\text{dom}f,\ \lambda\in[0,1]$, $f(\lambda x+(1-\lambda)y)\leq \max\lbrace f(x),f(y)\rbrace$, then $f$ is a quasiconvex function;

  • if $\forall x,y\in\text{dom}f,\ \exists m>0$ such that $f(y)\geq f(x)+\nabla f(x)^\top (y-x)+\dfrac{m}{2}||y-x||^2$, then $f$ is a strongly convex function.

Prop.

  • For a quasiconvex function, its sublevel set $S_\alpha:=\lbrace x\in\text{dom}f\ |\ f(x)\leq \alpha\rbrace$ is always convex.

  • The definition of strongly convex is equivalent to $f(x)-\dfrac{m}{2}x^2$ is convex.

Def. For a cone $C\subseteq \mathbb{R}^n$,

  • $P=\lbrace z\ |\ z^\top x\leq 0,\ x\in C\rbrace$ is called the polar cone of $C$;

  • $\mathcal{N}_C(x_0)=\lbrace z\ |\ z^\top(x-x_0)\leq 0,\ \forall x\in C\rbrace$ is called the normal cone of $C$ at the point $x_0\in C$;

  • the polar cone of normal cone at $x_0$ is called tangent cone.

Thm. (Optimality in differentiable case) For a differantiable convex function $f$ and a nonempty convex set $X$, we have

$$ x^*\in\mathop{\arg\min}\limits_{x\in X} \iff \nabla f(x^{*})^\top (x-x^{*})\geq 0, \forall x\in X $$

Note: If $\nabla f(x^*)=0$, then obviously $x^*$ is a local minimum.

If $\nabla f(x^*)\neq 0$, then $-\nabla f(x^{*})\in\mathcal{N}_C(x^{*})$, and $-\nabla f(x^*)$ denotes a supporting hyperplane of $X$.

Note: The FOC can be interpreted by geometric view.

Def. A vector $s_0$ is called the subgradient at $x_0$ if $f(x)\geq f(x_0)+s_0^\top(x-x_0)$.

Note that if $f(x)$ is differentiable at $x_0$, then the subgradient at $x_0$ is unique, namely the gradient. Otherwise, $\partial f(x_0)$ will be a set with infinite elements.

Thm. If $f(x)$ is convex, then $\partial f(x)$ is nonempty and bounded for all $x\in \text{intdom} f$.

The optimality condition is similar to the situation of differentiable functions.

Prop. The subgradient can be regarded as the most exact linear estimation of convex function $f(x)$, which is also called “oracle inequality”.
